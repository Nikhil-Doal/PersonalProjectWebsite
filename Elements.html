Elements

An HTML element is defined by a start tag, some content, and an end tag.

Elements are the building block of HTML

Here is a a typical HTML element

<element>Content goes here ...</element>

Elements are what we use to make code in HTML.

Different elements have different functions, these are what we will be learning about in the upcoming sections

Nested HTML Elements

We can nest HTML elements by putting elements inside of other elements

Here is an example. This is what we must do in each HTML document.

			<!DOCTYPE html>
			<html>
			    <head>
				    ...content...
			    </head>
			    <body>
				    ...content...
			    </body>
			</html>

It is very important to make sure that we end each tag that we open.

Empty HTML elements

There are some elements in HTML which are empty and do not need to be closed, an example of these would be the line break tag

The line break tag or <br> is a tag which adds a line break to the content

It does not need an end tag as it starts and ends itself

<p>This is a<br>sample line</p>
