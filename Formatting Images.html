Formatting Images

There's not really a lot to formatting images, and the process is very simple

Firstly we have to give the image a class or an id attribut so that we can call it in CSS

After we do that we can use the commands in CSS to format the image

These include the:

  • Height
  • Width
  • Border
  • Padding
  • Opacity
  • Position
  • etc.
