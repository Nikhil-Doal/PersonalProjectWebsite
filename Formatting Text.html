Formatting Text

Text Color

The color property is used to set the color of the text. The color is specified by:

  • a color name - like "red"
  • a HEX value - like "#ff0000"
  • an RGB value - like "rgb(255,0,0)"

For example

h1{color: red;}

This produces a red heading

Text Alignment

We can also align text using the text-align and vertical align commands

h1{text-align: center; vertical-align:center;}

This aligns the heading to the center of the page

Text font and fontsize

We can change the font family of text and its font size using the font-family and font-size commands

h1{font-size: 20px; font:family: arial;}

This changes the font size to twenty and the font family to arial
