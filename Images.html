Images

Images can improve the look of a website and can be very helpful

Here is an image

First we will learn about the Syntax of the image tag

Syntax

Here is the normal image tag

<img src="" alt="" width="" height="">

The image tag is self-closing like the hr and br tags

The src, alt, width and height are called attributes, they are the specifications that we give to the image tag

src

The src attribute is the source attribute, inside of this we put the source of our image, for example the url.

We can also copy the file path of the image on our computer and put it in the source

The file name MUST go into the quotation "" marks

alt

The alt attribute is fallback text incase the image does not load

It is important to have an alt attribute as it helps the viewer understand what an image would have shown

Width and Height

We can use the width and height elements to specify the width and heigh of an image

If one of them is left blank, it is set to auto, meaning it will preserve the image's ratio

If left blank the image will take up its original size
