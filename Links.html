Links

Let's now move onto links.

Links are very important in HTML as they allow us to create multiple pages and interconnect webpages

When you hover over a link, the mouse will become a little hand

Syntax

Here is the general syntax for a link

<a href="url">link text</a>

By default, links will appear as follows in all browsers:

  • An unvisited link is underlined and blue
  • A visited link is underlined and purple
  • An active link is underlined and red

The target Attribute

If you don't specify a target for a link, the linked page will be shown in the same browser window. However, if you want to display the linked page in a different window, you need to specify a different target.

Here is the syntax

<a href="url" target="">link text</a>

  • _self - Default. Opens the document in the same window/tab as it was clicked
  • _blank - Opens the document in a new window or tab
  • _parent - Opens the document in the parent frame
  • _top - Opens the document in the full body of the window

Using other things as links

We can use many other things as links as well

For example, images and buttons can be used as links

Images

<a href=""><img src=""></a>

We can do similar things with buttons, but for that we will need to know some JavaScript

We can also use links to mail people

Here is the syntax

<a href="mailto:someone@example.com">Send email</a>

This will allow us to email someone by clciking the link
