Lists and Tables

Lists

There are 2 types of lists in HTML

An Unordered HTML List

  • Coffee
  • Tea
  • Milk

Syntax:

<ul>

<li>Item name</li>

</ul>

An Ordered HTML List

  1. Coffee
  2. Tea
  3. Milk

Syntax:

<ol>

<li>Item name</li>

</ol>

Discription lists

HTML also supports description lists.

A description list is a list of terms, with a description of each term.

Coffee
- black hot drink
Milk
- white cold drink

Syntax:

<dl>

<dt>Item name</dt>

<dd>Item description</dd>

</dl>

Tables

A table in HTML consists of table cells inside rows and columns.

Here is a table

Person 1 Person 2 Person 3
John Mike Robert
16 14 10

Here is the syntax we need to use

<table>

<tr>

<th>Item</th>

</tr>

<tr>

<td>Item</td>

</tr>

</table>

There are 4 different tags that we have used here

The table tag defines a table

Then the tr tag defines a row

The th and td tags define elements, i.e the columns

We have two tags because the th is also known as the heading, and can be style differently in CSS
