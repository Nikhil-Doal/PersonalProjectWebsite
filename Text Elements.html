Headings

In HTML we have 6 types of headings

These headings start at h1 and continue to get smaller till h6

Here are the headings

Headings are IMPORTANT

Search engines use the headings to index the structure and content of your web pages.

Users often skim a page by its headings. It is important to use headings to show the document structure.

We can change the size of headings if we wish by using CSS

Paragraps and the <p> tag

We can define a paragraph using the <p> tag.

A new paragraph tag will always start on a new line.

<p>This is a sample paragraph</p>

<p>This is a new line</p>

This is the result

This is a sample paragraph

This is a new line

We cannot add multiple lines of text or multiple spaces into one <p> tag, the browser will condense all spaces to one space and all lines to one line

To solve this we can use the <pre> tag. This tag, also known as the preformatted tag lets us keep the formatting of our text

Inside the pre tag we can write with whatever formatting we wish and that will be saved

The Horizontal Rule and the Line Break

In HTML we have a few ways to split text.

For this we can use the <hr> and <br> tags.

<hr>

We use the horizontal rule to create a line that splits a page

This can be used to create sections or split text

Here's an example

<p>This is some text</p>

<hr>

<p>This is some more text</p>

This is the result

This is some text

This is some more text

<br>

We use the line break to create a break a line of text

It can be used to split longer lines of text into shorter and smaller lines

Here's an example

<p>This is<br>some text</p>
